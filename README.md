# 📝 TaskFlow

## 📋 Overview
**TaskFlow** is a simple and intuitive task management web application designed to help users efficiently organize their daily activities.  
It allows users to create, manage, and track tasks seamlessly — whether they are **pending**, **in progress**, or **completed**.

---

## 🚀 Features
- ✅ Add, edit, and delete tasks  
- 📅 Categorize tasks by **pending**, **upcoming**, and **completed**  
- 🔍 Search and filter tasks easily  
- 🎨 Responsive and user-friendly interface  
- 💾 Data stored securely in **MongoDB**  
- 🔐 Secure user authentication with email and password  

---

## 🛠️ Tech Stack
**Frontend:**  
- HTML5  
- CSS3  
- JavaScript  

**Backend:**  
- Node.js  
- Express.js  
- MongoDB (with Mongoose ODM)
